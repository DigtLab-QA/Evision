Руководство пользователя программного обеспечения eVision

eVision – программное обеспечение для организации системы видеонаблюдения со встроенной событийной аналитикой, основанной на классификации объектов с последующей их реидентификацией. 
Типы детектируемых объектов: 
силуэт человека
лицо
транспортное средство
автомобильный номер
Кроме встроенной видеоаналитики, eVision умеет обрабатывать данные с устройств со встроенной аппаратной аналитикой, синхронизировать базу пользователей и журнал событий. 
Сферы применения: 
Для организации системы видеонаблюдения и записи архива по детектору движения;
Для организации бесконтактной системы контроля и управления доступом с использованием биометрической идентификации;
Для организации СКУД  с использованием идентификации автотранспорта по номеру авто;
Для трекинга посетителей, формирования отчетов по посещаемости.
